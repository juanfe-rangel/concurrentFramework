🚀 MicroSpringboot Web Server

Bienvenido al Servidor Web

Este es un servidor web minimalista construido en Java con capacidades IoC (Inversión de Control).

El servidor demuestra las capacidades reflexivas de Java para cargar POJOs y construir aplicaciones web.

Endpoints Disponibles

Controladores REST:

  • / - Página principal
  • /hello - Saludo simple
  • /pi - Valor de PI
  • /greeting - Saludo con parámetro (sin parámetro)
  • /greeting?name=Juan - Saludo personalizado

Archivos Estáticos

El servidor soporta archivos HTML y PNG:

  • /index.html - Esta página
  • /test.png - Imagen de prueba

Características Implementadas

  • ✅ Anotaciones @RestController y @GetMapping
  • ✅ Soporte para @RequestParam con valores por defecto
  • ✅ Carga de controladores desde línea de comandos
  • ✅ Escaneo automático del classpath
  • ✅ Servidor HTTP para múltiples solicitudes no concurrentes
  • ✅ Servir archivos HTML y PNG estáticos
  • ✅ Framework IoC basado en reflexión

Ejemplos de Uso

Modo 1: Cargar controladores específicos

java -cp target/classes org.example.MicroSpringboot org.example.HelloController org.example.GreetingController

Modo 2: Escaneo automático del classpath

java -cp target/classes org.example.MicroSpringboot --scan
